★説明
	・一つのサーバーと複数のクライアント(現在最大2人に設定)の間でやり取りするチャット
	・サーバー側で[接続受付]を押した後、クライアント側でサーバアドレスを設定(デフォルトのままでもOK)し、接続要求を押して、問題がなければ
　	　通信開始
	・送信メッセージの内容を[送信]を押すことで、サーバを経由して全クライアントに送られる
	・[切断]で通信終了

★制作期間
	約三か月

★制作意図
	授業の一環で、通信についての学習のために制作しました。

	デフォルトで用意されるように実装しました。
